# Color Scheme Generator

Generate beautiful colour palettes from any base colour using the [The Color API](https://www.thecolorapi.com). Pick a colour, choose a scheme mode, and get an instant 5-colour palette.

## Features

- Pick any base colour using a native colour picker
- 8 scheme modes: Monochrome, Analogic, Complement, Triad, Quad and more
- Instant palette generation via The Color API
- Click any colour swatch to copy its hex value to clipboard

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- [The Color API](https://www.thecolorapi.com)

## How to Run Locally

```bash
# Clone the repo
git clone https://github.com/Bindekere/Color-scheme-generator.git

# Open index.html in your browser
```

No dependencies or build step needed — just open `index.html`.
